# Integrated Paper
Ever REALLY wanted FAWE in singleplayer?

## Setup
Place a valid `server.jar` in your instance's folder at startup.
Then, launch the game and simply enter a singleplayer world.
No configuration necessary, just start playing Paper!

---
|  | Location relative to client root |
| :---: | :---: |
| **Server Root** | server/ |
| **Server Jar** | server.jar *(Must be 26.1 or higher!)* |
| **World** | <unchanged> |
| **Java** | <unchanged> |

| Server property | Automatic effect by Integrated Paper |
| :---: | :---: |
| **OPs** | Includes yourself as an OP automatically |
| **Server properties** | Offline mode for offline players |
---
